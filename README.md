# Laboratorio Virtual: Plano Inclinado

Simulador interactivo para el estudio de la física del movimiento en planos inclinados con y sin sistema de poleas.

## Características

- Simulación en tiempo real del movimiento en plano inclinado
- Sistema básico y sistema con poleas
- Visualización gráfica del experimento
- Exportación de datos a Excel
- Historial de configuraciones
- Explicaciones teóricas y fórmulas
